# Social Media Engagement Analytics Dashboard

## 📊 Project Overview

This project analyzes social media engagement metrics across different platforms such as Instagram, Facebook, and Twitter.
The dashboard helps track likes, comments, shares, and engagement rate to understand content performance.

## 🎯 Project Goal

To analyze social media engagement data and identify patterns in user interaction, content performance, and best posting times.

## 🛠 Tools Used

* Power BI
* Data Visualization
* Data Analysis
* CSV Dataset

## 📂 Dataset

The dataset contains engagement metrics including:

* Platform
* Post Type
* Likes
* Shares
* Comments

## 📈 Dashboard Insights

* Instagram shows the highest engagement compared to other platforms.
* Video posts generate the most likes and interactions.
* Carousel posts also perform well.
* Best posting time identified between **6 PM – 9 PM**.

## 📊 Dashboard Preview

The dashboard visualizes:

* Total likes, comments, and shares
* Engagement by platform
* Engagement by post type
* Overall engagement distribution

## 🚀 Project Outcome

This dashboard helps understand which platform and content type generate the most engagement and suggests the best posting strategy for better reach.
